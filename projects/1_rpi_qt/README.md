# dev_350g

This directory contains documentation and notes related to my work on the **dev_350g** project at Novin Med Corp.

## Project Overview

**dev_350g** is an embedded device project developed using Qt/QML for the user interface. The project targets ARM-based hardware and includes features such as multi-language support and device control.

## My Contributions

- Added Russian translation for the device UI
- Set up cross-compilation environment using Linaro toolchain and Raspberry Pi sysroot
- Patched Qt modules for compatibility with Ubuntu 22.04
- Fixed build issues and improved code portability

## Development Environment

- **OS:** Pop!_OS (in VMware VM)
- **Tools:** Qt, QML, CMake, qmake, Linaro toolchain, Git
- **Target:** ARM (Raspberry Pi)
